# Rock Paper Scissors Game
A simple Rock-Paper-Scissors game built using **HTML, CSS, and JavaScript**.

## How to Play
- Select Rock, Paper, or Scissors.
- The computer will make a random choice.
- Rules:
  - Rock beats Scissors
  - Scissors beats Paper
  - Paper beats Rock
